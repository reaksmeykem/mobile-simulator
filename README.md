# Mobile Simulator

A Chrome extension to simulate mobile devices and test responsive designs with realistic device mockups.

<!-- ![Screenshot](screenshot.png) -->

## Features

- **16+ Devices** — iPhone, Samsung Galaxy, Google Pixel, iPad, MacBook
- **Realistic Mockups** — Dynamic Island, punch-hole camera, bezels, physical buttons
- **Rotate** — Switch between portrait and landscape
- **Theme-aware Status Bar** — Matches the website's background color
- **Persistent State** — Simulation survives page navigation

## Installation

1. Clone this repo
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (top right toggle)
4. Click **Load unpacked** and select the project folder
5. Open any website and click the extension icon

> **Note:** The extension only works on regular websites. It does not work on `chrome://`, `edge://`, or browser internal pages.

## Usage

1. Navigate to any website
2. Click the Mobile Simulator icon in the toolbar
3. Select a device from the dropdown
4. Click **Apply Simulator**
5. Use **Rotate** / **Show Frame** / **Close** as needed
