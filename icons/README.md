# Icons Folder

This folder needs three PNG icon files for the Chrome extension to work:

- `icon16.png` (16×16 pixels)
- `icon48.png` (48×48 pixels)  
- `icon128.png` (128×128 pixels)

## How to Generate Icons

Go back to the `mobile-simulator` folder and use one of these methods:

### Option 1: Auto-Download (Easiest)
Open `generate-icons-simple.html` in your browser - icons will auto-download

### Option 2: Python Script
Run `generate-icons.bat` (Windows) or `python generate_icons.py`

### Option 3: Manual Download
Open `create-icons.html` and click "Download All Icons"

After generating, the PNG files should be placed in this folder.
